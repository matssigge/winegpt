Implement this feature using a test-first approach.

Process:
1. Restate the feature briefly.
2. Identify the smallest useful vertical slice.
3. Add or update tests first.
4. Implement the minimal code to pass.
5. Refactor only if it simplifies the result.
6. Run focused tests, then broader relevant tests.
7. Summarize what changed, what is covered, and what remains.

Constraints:
- Keep scope narrow.
- Do not refactor unrelated code.
- Avoid duplication.
- Avoid speculative abstractions.
